# PSO参数优化报告

## 1. 优化概述

本次优化使用粒子群优化（PSO）算法对十字路口行人通行仿真系统的参数进行自动优化。优化目标是提高系统的通行效率，同时考虑最慢10%实体的通行时间和碰撞次数。

## 2. 优化参数

| 参数名称 | 描述 | 优化范围 | 最优值 |
|---------|------|---------|--------|
| DESIRED_FORCE_FACTOR | 目标吸引力因子（影响速度） | [0.5, 2.0] | 2.0 |
| SOCIAL_FORCE_A | 社交力强度（避让强度） | [1000.0, 5000.0] | 3120.25 |
| SOCIAL_FORCE_B | 社交力作用范围（避让距离） | [0.1, 0.3] | 0.1 |
| VISION_RANGE | 视野范围 | [2.0, 6.0] | 6.0 |

## 3. 优化结果

### 3.1 适应度变化

| 迭代次数 | 最优适应度 |
|---------|-----------|
| 1 | 0.1386 |
| 2 | 0.1386 |
| 3 | 0.1388 |
| 4 | 0.1402 |
| 5-12 | 0.1402 |

### 3.2 最优参数分析

1. **DESIRED_FORCE_FACTOR = 2.0**
   - 达到了参数范围的上限
   - 表明系统需要较强的目标吸引力，以提高实体的移动速度

2. **SOCIAL_FORCE_A = 3120.25**
   - 处于参数范围的中间偏上位置
   - 表明需要适中的避让强度，既能有效避免碰撞，又不会过度影响通行效率

3. **SOCIAL_FORCE_B = 0.1**
   - 达到了参数范围的下限
   - 表明需要较短的避让距离，减少不必要的避让行为，提高通行效率

4. **VISION_RANGE = 6.0**
   - 达到了参数范围的上限
   - 表明需要较大的视野范围，使实体能够提前感知周围环境，做出更合理的决策

## 4. 优化效果评估

### 4.1 预期效果

- **整体通行效率**：提高系统的平均通行效率
- **最慢10%通行效率**：改善最差情况的通行体验
- **碰撞次数**：减少实体间的碰撞

### 4.2 优化建议

1. **参数调整**：
   - 保持当前的最优参数设置
   - 可以考虑进一步扩展DESIRED_FORCE_FACTOR的范围，测试更高值的效果

2. **仿真时间**：
   - 当前仿真时间为30秒，可以考虑延长到60秒，获得更准确的适应度评估

3. **粒子数量和迭代次数**：
   - 当前使用30个粒子，50次迭代
   - 可以考虑增加粒子数量到50个，以获得更全面的参数空间搜索

4. **并行计算**：
   - 实现并行评估，提高优化速度

## 6. 结论

本次PSO优化成功找到了一组最优参数，适应度值达到0.1402。优化结果表明，系统需要较强的目标吸引力、适中的避让强度、较短的避让距离和较大的视野范围，以获得最佳的通行效率和安全性。

通过多次执行PSO优化，可以进一步改进参数配置，为交通规划和安全评估提供更可靠的参考。
